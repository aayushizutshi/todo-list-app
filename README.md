# To-Do List App 📝

A simple and functional To-Do List web application built using HTML, CSS, and JavaScript.

## Features
- Add tasks to your list
- Tasks persist after page refresh (Local Storage)
- Empty input validation
- Clean and responsive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM, Local Storage)

## How to Use
1. Type your task in the input box
2. Click "Add" button
3. Task will be saved even after refresh!
